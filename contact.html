📬 Как с нами связаться

Лучший способ связаться — через GitHub Issues (для технических вопросов и багов) или через Ko-fi (для общих вопросов и поддержки).

Мы читаем все сообщения и стараемся отвечать как можно быстрее.

Обычное время ответа: 1-3 дня

🐛 Нашли ошибку?

Если что-то не работает — пожалуйста, опишите:

  • Что именно не работает
  • На каком устройстве и браузере
  • Шаги для воспроизведения проблемы
Сообщайте об ошибках через GitHub

📬 How to Contact Us

The best way to reach us is through GitHub Issues (for technical questions and bugs) or Ko-fi (for general questions and support).

We read all messages and try to respond as quickly as possible.

Typical response time: 1-3 days

🐛 Found a Bug?

If something isn't working, please describe:

  • What exactly isn't working
  • Your device and browser
  • Steps to reproduce the issue
Report bugs via GitHub

💡 Have an Idea?

Want a new tool or improvement to an existing one? Write to us — we consider all suggestions! Many features were added based on user requests.
